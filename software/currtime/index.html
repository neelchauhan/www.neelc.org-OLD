currtime

currtime is an enhanced command-line operated real time clock. currtime displays the current system date and time in a Unix-compatible format.

currtime is written in C and is licensed under the Simplified BSD License.

Features

  • Displays time in Unix-compatible format
  • Shows the time on the same line as default
  • Can run for a specified number of ticks (seconds) with the -T (ticks) flag
  • Option to display each tick on a new line with the -n flag

Usage

The usage information can be viewed here and is also included with the currtime distribution in the file README.
