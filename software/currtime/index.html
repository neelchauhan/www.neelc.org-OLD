currtime

currtime is a command-line operated real time clock. currtime displays the current system date and time in a Unix-compatible format.

currtime is written in C and is licensed under the Simplified BSD License.

Download

You can download currtime from the /files/currtime/0.1/ directory, or look at the source code through cgit.

To view the usage information, it is located at /files/currtime/README and is also included with the currtime distribution.
